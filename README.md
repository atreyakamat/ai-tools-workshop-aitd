# ai-tools-workshop-aitd

AITD AI Tools Hands-On Workshop

## SOLO × AITD AI Build Workshop — Presentation

A React-based interactive slideshow presentation for the **SOLO × AITD AI Build Workshop**.

### Features

- 🎨 **Dark futuristic theme** — black / deep purple / navy background with neon blue / violet / cyan accents
- 📊 **34 slides** covering the full workshop curriculum
- ⌨️ **Keyboard navigation** — use arrow keys to move between slides
- 🖱️ **Click navigation** — Prev / Next buttons and dot indicators
- 📱 **Responsive** — works on all screen sizes

### Slide Sections

| Slides | Section |
|--------|---------|
| 1 | Cover — SOLO × AITD AI Build Workshop |
| 2–5 | SOLO Platform — About, Roadmap, Onboarding, Demo |
| 6–8 | Development Pathways — Frontend, Backend, Why Paths Matter |
| 9–13 | Prompt Engineering — Role, Context, Objective, Constraints |
| 14–18 | Google Stitch — UI generation tool |
| 19–23 | Google AI Studio — Gemini model playground |
| 24–28 | Antigravity — AI automation platform |
| 29–33 | Gemini CLI — Terminal-based AI development |
| 34 | Final — Build Mode / Call to Action |

### Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

### Navigation

- **Arrow keys** (← →) — navigate between slides
- **Section buttons** in the header — jump to any section
- **Dot indicators** in the footer — click any dot to jump to that slide
- **Prev / Next buttons** — step through slides one at a time
